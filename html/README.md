# Leo's Learn & Play - Web App

This is the web version of the Child Game, designed for kids to learn alphabets, animals, and fruits in a fun, interactive way.

## Features
- **Interactive Tiles**: High-quality emojis and text.
- **Voice Support**: Uses Web Speech API for Text-to-Speech.
- **Animations**: Powered by Animate.css and Canvas Confetti.
- **Mobile Optimized**: Responsive design using Bootstrap 5.
- **PWA Ready**: Includes `manifest.json` and a high-quality icon.

## How to upload to Indus Appstore
1. **Zip the contents**: Select all files in this `html` folder (`index.html`, `style.css`, `game.js`, `game_data.js`, `manifest.json`, `icon.png`) and create a `.zip` file.
2. **Indus Developer Console**: Log in to your [Indus Appstore Developer Console](https://developer.indusappstore.com/).
3. **Add New App**: Select "Web App" or "PWA" as the type.
4. **Upload Zip**: Upload the zip file you created.
5. **Fill Details**: Provide the app name, description, and icons as required.

## Local Preview
To see the app in action locally:
1. Open a terminal.
2. Run a local server (e.g., `npx serve .` or `python -m http.server`).
3. Open `http://localhost:port` in your browser.
